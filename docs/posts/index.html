ObjectInteraction  [draft]

** Implementation of Object Interaction** There are three object interaction tasks within the parcours. Each spawns two T-shaped objects in random position and rotation. One is slightly transparent, the other is opaque. The users have to rotate and translate the position of the opaque object to make it fit the transparent ones. In the end, the accuracy is measured by calculating the offset of the different parts of the transparent object from the opaque one....
